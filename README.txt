Draws spirograph in OpenGL window.

Usage:  ./spiro -r1 <senary number> -r2 <senary number> -op <senary number> -sp <senary number> -cl <b/g/r/y/p/w>


-r1: radius 1 value must be between 0 and 1054(base 6)
-r2: radius 2 value must be between 0 and 1054(base 6)
-op:  offset position value must be between 1 and 1054(base 6)
-sp: speed value must be between 1 and 244(6)
-cl: color must be b(blue), g(green), r(red), y(yellow), p(pink), w(white).
